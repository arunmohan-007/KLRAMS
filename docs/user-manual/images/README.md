# User-manual screenshots

Drop the screenshot PNGs into this folder using the exact filenames listed in
**Appendix D — Figure checklist** of `../KLRAMS-User-Manual.md`
(e.g. `fig-04-07-nsv-video-sync.png`). Each figure in the manual references its
file by that name, so the images appear automatically — no edits to the manual
are needed.

Recommended capture settings: 1440×900 browser window, signed in as
SUPER_ADMIN, with representative data loaded. Crop out personal bookmarks/tabs.
