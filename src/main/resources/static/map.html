Pavement Condition Index

PCI — IRC:82

Set the six distress weightages and generate the index, or open the detailed report. The map is hidden while you work here.
PCI = Σ ( wi × Ii ) · six indices · 0–100 scale
Weightage factor (editable)Σ weights = 1.00
PCI rating scale (Table 5.5)
Generate PCI produces two layers — Composite PCI (area-weighted distress) and Worst-Lane PCI (maximum distress per parameter). Each parameter’s distress is converted to a 0–100 index; thresholds follow IRC:82 Table 5.3 (editable under Road Condition). Default weightages are IRC:82 Table 5.4 (Σ = 1.00).
