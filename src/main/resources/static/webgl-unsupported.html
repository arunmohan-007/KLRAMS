The map needs graphics acceleration turned on

The full KLRAMS GIS viewer draws the road network using WebGL, which is currently disabled in this browser on this computer. You can still use the Lite map right now (it works without WebGL) — or enable WebGL below to unlock the full viewer.

  1. Open a new tab, type chrome://settings/system and press Enter. Turn "Use graphics acceleration when available" ON, then click Relaunch.
  2. If it's still off, open chrome://flags/#ignore-gpu-blocklist, set it to Enabled, and click Relaunch.
  3. Make sure the computer's graphics driver is up to date (ask IT if unsure).
  4. Reopen KLRAMS and click Try the map again below.

Managing many PWD computers? IT can enable this for everyone at once with a Chrome/Edge group policy: set HardwareAccelerationModeEnabled = 1 (Enabled). Then no per-machine change is needed.
