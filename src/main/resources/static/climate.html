Classification & Index Values

Each band maps a raw value to an index (10–100). Use Source Class for categorical layers (rainfall / flood) or Min/Max for ranges (slope). Leave Max blank for an open-ended top band.

Combined Climate Vulnerability Index (CVI)

CVI = (FSI × wFSI) + (RFI × wRFI) + (SLI × wSLI). Weights drive the calculation; the formula text is for reference.

Index Generation

Seed Grid creates one climate row per chainage segment. Recalculate re-derives RFI/SLI/FSI → CVI → category from stored raw values (cheap; no raster). Raw values are filled by the PostGIS extraction in build 77.
